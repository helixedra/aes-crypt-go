AES Cryptor v0.1.1

О программе

AES Cryptor - это утилита командной строки для безопасного шифрования и дешифрования файлов с использованием алгоритма AES-256-GCM. Обеспечивает военный уровень защиты ваших данных.

Возможности:

  • Надежное шифрование AES-256-GCM
  • Безопасная генерация ключей
  • Кросс-платформенная поддержка (Windows, macOS, Linux)
  • Простой и интуитивно понятный интерфейс
  • Не требует дополнительных зависимостей

Системные требования

  • Windows 10/11 (64-бит), macOS 10.15+ или Linux (64-бит)
  • Не менее 10 МБ свободного места на диске

Проверка целостности

# Linux/macOS
shasum -a 256 имя-файла.zip

# Windows (PowerShell)
Get-FileHash -Algorithm SHA256 -Path "имя-файла.zip"

Сравните вывод с указанной выше контрольной суммой.
