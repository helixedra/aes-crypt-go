AES Cryptor v0.1.1

AES Cryptor is a command-line utility for secure file encryption and decryption using the AES-256-GCM algorithm.

Features:

  • Secure AES-256-GCM encryption
  • Secure key generation
  • Cross-platform support (Windows, macOS, Linux)
  • No additional dependencies required

Windows 10/11 (64-bit), macOS 10.15+, or Linux (64-bit)
At least 10 MB of free disk space

Linux/macOS
shasum -a 256 filename.zip

Windows (PowerShell)
Get-FileHash -Algorithm SHA256 -Path "filename.zip"

Compare the output with the checksum provided above.
