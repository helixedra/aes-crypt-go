AES Cryptor

About

AES Cryptor is a command-line utility for secure file encryption and decryption using the AES-256-GCM algorithm. It provides military-grade encryption to protect your sensitive files.

Features:

  • Strong AES-256-GCM encryption
  • Secure key generation
  • Cross-platform support (Windows, macOS, Linux)
  • Simple and intuitive interface
  • No external dependencies

System Requirements

  • Windows 10/11 (64-bit), macOS 10.15+, or Linux (64-bit)
  • At least 10MB free disk space
